    **Dollar detection rules (from spec CONV-01):**
    - A call expression triggers QRL extraction if:
      1. Callee is `$` or ends with `$` (e.g., `component$`, `useTask$`)
      2. Callee resolves to an import from `@qwik.dev/core` (check GlobalCollect.imports) OR callee is `$` and is a known marker
      3. First argument is a function expression or arrow function expression
    - Special cases: `sync$` triggers CONV-13 (sync serialization, not QRL extraction). Detect and flag separately.
    - `component$` is syntactic sugar: `component$(fn)` -> `componentQrl(qrl(fn))` with PURE annotation
