    Add the following Traverse trait hooks to the `impl<'a> Traverse<'a, ()> for QwikTransform<'a>` block in transform.rs. These replicate SWC's stack_ctxt push/pop behavior for descriptive symbol naming.

    **1. Variable declarator (enter/exit):**
    ```rust
    fn enter_variable_declarator(
        &mut self,
        node: &mut VariableDeclarator<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        if let BindingPattern::BindingIdentifier(ident) = &node.id {
            self.stack_ctxt.push(ident.name.to_string());
        }
    }

    fn exit_variable_declarator(
        &mut self,
        node: &mut VariableDeclarator<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        if matches!(&node.id, BindingPattern::BindingIdentifier(_)) {
            self.stack_ctxt.pop();
        }
    }
    ```

    **2. Function declaration (use enter_declaration/exit_declaration):**
    Note: OXC's `enter_function` is called for both function declarations AND function expressions. For stack_ctxt, we only want function DECLARATIONS (named functions like `function foo() {}`). Use `enter_declaration` / `exit_declaration` matching `Declaration::Function(func)` where `func.id` is Some.
    ```rust
    fn enter_declaration(
        &mut self,
        node: &mut Declaration<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        match node {
            Declaration::Function(func) => {
                if let Some(id) = &func.id {
                    self.stack_ctxt.push(id.name.to_string());
                }
            }
            Declaration::Class(class) => {
                if let Some(id) = &class.id {
                    self.stack_ctxt.push(id.name.to_string());
                }
            }
            _ => {}
        }
    }

    fn exit_declaration(
        &mut self,
        node: &mut Declaration<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        match node {
            Declaration::Function(func) => {
                if func.id.is_some() {
                    self.stack_ctxt.pop();
                }
            }
            Declaration::Class(class) => {
                if class.id.is_some() {
                    self.stack_ctxt.pop();
                }
            }
            _ => {}
        }
    }
    ```

    **5. Export default declaration (push file_stem):**
    The OXC transform has `self.file_name` available (set during construction). For export default, push the file stem (filename without extension). SWC uses `file_stem`, and for `index` files uses the parent folder name. For this implementation, push the file stem derived from `self.file_name`:
    ```rust
    fn enter_export_default_declaration(
        &mut self,
        _node: &mut ExportDefaultDeclaration<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        // Push file stem (e.g., "test" from "test.tsx") for export default context
        let file_stem = self.file_name
            .rsplit('.')
            .last()
            .unwrap_or(&self.file_name)
            .to_string();
        // For "index" files, try to use the parent dir name instead
        // But since we only have file_name (not full path), just use the stem
        self.stack_ctxt.push(file_stem);
    }

    fn exit_export_default_declaration(
        &mut self,
        _node: &mut ExportDefaultDeclaration<'a>,
        _ctx: &mut TraverseCtx<'a, ()>,
    ) {
        self.stack_ctxt.pop();
    }
    ```
