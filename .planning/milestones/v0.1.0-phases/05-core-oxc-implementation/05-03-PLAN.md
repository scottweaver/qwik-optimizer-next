    **parse.rs** -- Create a wrapper around OXC's Parser and SemanticBuilder:
    ```rust
    pub(crate) struct ParseResult<'a> {
        pub program: Program<'a>,
        pub scoping: Scoping,
        pub source_text: &'a str,
        pub panicked: bool,
    }

    pub(crate) fn parse_module<'a>(
        allocator: &'a Allocator,
        source: &'a str,
        source_type: SourceType,
    ) -> ParseResult<'a> {
        let ret = Parser::new(allocator, source, source_type).parse();
        let scoping = SemanticBuilder::new()
            .with_cfg(false)
            .build(&ret.program)
            .semantic
            .into_scoping();
        ParseResult {
            program: ret.program,
            scoping,
            source_text: source,
            panicked: ret.panicked,
        }
    }
    ```
    Check OXC 0.123 API -- `SemanticBuilder` may have changed method names since 0.113. Verify by reading docs.rs or Jack's code. The key is to get a `Scoping` object that provides `ScopeTree`, `SymbolTable`, and `ReferenceTable`.

    Note: The `Program` is arena-allocated and lives as long as `allocator`. The `Scoping` is owned data that can outlive the parse. Check whether OXC 0.123 `SemanticBuilder::build()` returns `SemanticBuilderReturn` or has changed the API.

    **collector.rs** -- Implement GlobalCollect per spec Section 2 (Stage 2):
    ```rust
    pub(crate) struct GlobalCollect {
        pub imports: IndexMap<String, ImportInfo>,
        pub exports: IndexMap<String, ExportInfo>,
        pub root: IndexMap<String, Span>,
        // Reverse lookup: (source, specifier) -> local_name
        rev_imports: HashMap<(String, String), String>,
    }

    pub(crate) struct ImportInfo {
        pub source: String,
        pub specifier: String,  // "default", "*", or the exported name
        pub local_name: String,
        pub kind: ImportKind,    // Value, Type, or TypeOnly
    }
    ```

    Walk `program.body` statements:
    - `ImportDeclaration`: Catalog each specifier. Track namespace imports (`* as`), default imports, and named imports. Record the source module. For type-only imports (`import type { X }`), mark as TypeOnly.
    - `ExportNamedDeclaration`: If has declaration, extract bindings. If has specifiers (re-exports), record source.
    - `ExportDefaultDeclaration`: Record as "default" export.
    - Other statements: Extract root-level binding names (VariableDeclaration, FunctionDeclaration, ClassDeclaration) and record in `root` with their spans.

    Implement `GlobalCollect::is_import(&self, name: &str) -> bool` and `GlobalCollect::get_import(&self, name: &str) -> Option<&ImportInfo>`.

    Update `lib.rs` to add `pub(crate) mod parse; pub(crate) mod collector;`.
